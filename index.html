01 / PROJECT

MeshSOS

Engineered a native Android app for decentralized P2P emergency communication in off-grid environments using Bluetooth Low Energy (BLE).

Built entirely without internet dependency, MeshSOS creates a peer-to-peer mesh network between nearby Android devices using BLE beacons. It routes SOS signals, GPS coordinates, and text messages across multiple hops — each device acting as a relay node. Integrated Firebase for cloud sync when connectivity is eventually restored, ensuring no message is ever lost. Designed for disaster scenarios, remote expeditions, and grid-down emergencies.

02 / PROJECT

Smart City IAQ Monitor

Developed a full-stack IoT platform to centralize indoor air quality monitoring, featuring real-time data sync with microcontrollers.

A comprehensive IoT ecosystem spanning hardware to cloud. ESP32 microcontrollers sample CO2, particulate matter (PM2.5), humidity, and temperature at 5-second intervals. Data streams over WebSockets to a Node.js backend, where it's stored and served to a React dashboard featuring live graphs, threshold alerts, and historical trend analysis. Designed to scale across multiple rooms or floors of a smart building with a unified management console.
